# ☕ Lofi Cafe — 作業用カフェアプリ

カフェの雰囲気の中で作業に集中できる、シングルページWebアプリです。
NPCたちと同じ空間で、タスク管理・ポモドーロタイマー・BGMティッカーを楽しめます。

---

## 🌟 主な機能

### 🕐 時間帯テーマ
時刻に応じて自動的にテーマが切り替わります。ページを開いたままでも自動で反映されます。

| 時間帯 | 時間 | 雰囲気 |
|--------|------|--------|
| 🌅 朝カフェ | 5:00〜11:00 | 爽やかな水色 |
| ☀️ 昼カフェ | 11:00〜17:00 | 温かみのあるベージュ |
| 🕯️ 深夜カフェ | 17:00〜5:00 | 落ち着いたダーク |

### 🗺️ フロアマップ
カフェ内の席をクリック／タップして自分の座席を選べます。NPCたちもリアルタイムで動き回ります。

- テーブルA〜D、ソファE、カウンター席
- NPCの位置がアイコンで表示される
- 席移動時にポップアップ通知

### 💬 チャットルーム
NPCたちが時間帯に応じた会話やひとりごとを自動でつぶやきます。自分もメッセージを送れます。

- メンバーチップ／チャットアイコンをタップするとNPCの自己紹介カードが表示される
- 朝・昼・夜それぞれ異なる会話パターン

### 🎵 音楽ティッカー
時間帯に合わせたBGMタイトルが横スクロールで流れます（実際の音楽再生はありません）。

### ⏱️ ポモドーロタイマー
- 作業25分 / 休憩5分のサイクル
- スタート・リセット・休憩ボタン

### ✅ タスク管理
- テキスト入力でタスクを追加（Enterキー対応）
- チェックで完了、削除ボタンあり
- 今日の進捗カウンター

### 🛎️ 注文システム
チャット入力欄左のベルアイコンから注文できます。店員さんが持ってきてくれます。

## 🛠️ 技術仕様

- **構成**: シングルHTMLファイル（HTML / CSS / JavaScript）
- **外部依存**: なし（フォントはシステムフォント使用）
- **対応**: スマートフォン・PC（最大幅720px）
- **ホスティング**: GitHub Pages対応

---

## 🚀 使い方

1. `cafe.html` をブラウザで開く
2. フロアマップの席をクリックして着席
3. タスクを追加して作業開始
4. タイマーをスタート！
